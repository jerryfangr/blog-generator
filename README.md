# 博客代码备份



## 配置

* **软件: [Hexo](https://hexo.io/)@5.2.0**
* **主题: [hexo-theme-yilia](https://github.com/litten/hexo-theme-yilia)**



## 使用

### 创建

```bash
hexo new blogName
```

### 生成html

```bash
hexo g
```

### 本地预览(可选)

```bash
hexo s
hexo s -p 8888
```

### 上传(github/your server/...)

```bash
hexo d
```
